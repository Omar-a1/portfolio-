Introduction to some languages that I have learned

Html(HyperText Markup Language):

It is the standard markup language for creating web pages. It describes the structure of a webpage using elements and tags.

CSS(Cascading Style Sheets):

It is a style sheet language used for describing the presentation of a document written in HTML or XML. It controls the layout and appearance of web pages.

JavaScript:

It is a high-level, dynamic, untyped, and interpreted programming language. It is widely used for creating interactive effects within web browsers.

Python:

It is a high-level, interpreted programming language known for its readability and simplicity. It is widely used in various fields, including web development, data analysis, artificial intelligence, and more.
